Belajar HTML

Hallo guys kali ini kita akan belajar mengenai apa itu html. HTML (Hypertext Markup Language) adalah standar dari markup language yang sering digunakan oleh WEB developer karena kemudahannya dalam pengaturan tata letak dan bahasa yang sangat sederhana dan mudah dipahami oleh seorang seorang yang baru belajar bahasa pemrogramman sekalipun. bahasa sangat cocok untuk menjadi bahasa pertama yang harus dipelajari jika ingin menjadi seorang WEB develop karena bahasa yang simpel dan mudah dimengerti.

  • Sejarah Singkat

    •

    HTML pertama kali diperkenalkan oleh Tim Berners-Lee pada tahun 1991. HTML 1.0 merupakan versi pertama yang diperkenalkan secara resmi pada tahun 1995. Sejak itu, HTML telah mengalami beberapa perubahan dan evolusi. Versi terbaru pada saat pengetahuan saya dipotong pada Januari 2022 adalah HTML5, yang memperkenalkan berbagai fitur baru dan perbaikan dari versi sebelumnya.


  • Kelebihan Html

    •

    Berikut adalah kelebihan dari HTML:

    1. Sederhana dan Mudah Dipahami: HTML dirancang untuk menjadi bahasa yang sederhana dan mudah dimengerti. Ini membuatnya ideal untuk pemula yang belajar pemrograman web.
    2. Kompatibilitas Tinggi: HTML dapat diakses di berbagai platform dan browser, menjadikannya bahasa yang sangat kompatibel untuk pengembangan web.
    3. Gratis dan Terbuka: HTML adalah standar terbuka yang bebas digunakan dan diakses oleh siapa pun. Ini membuatnya mudah diakses dan didistribusikan.
    4. Integrasi dengan CSS dan JavaScript: HTML dapat dengan mudah diintegrasikan dengan bahasa pemrograman lain seperti CSS (Cascading Style Sheets) untuk tata letak dan styling, serta JavaScript untuk membuat halaman web interaktif.
    5. Mendukung Multimedia: HTML5 membawa perbaikan besar dalam mendukung multimedia seperti audio dan video tanpa perlu plugin pihak ketiga.

  • Web Yang menggunakan HTML

    •
    • Google: Meskipun menggunakan teknologi web yang lebih kompleks, Google masih bergantung pada HTML untuk struktur dasar halaman web mereka.
    • Wikipedia: Wikipedia, sebagai sumber informasi online terkenal, menggunakan HTML untuk menyajikan kontennya dengan tata letak yang terstruktur.
    • GitHub: Platform pengembangan perangkat lunak ini menggunakan HTML untuk struktur halaman webnya.
    • Facebook: Meskipun menggunakan berbagai teknologi web, Facebook tetap mengandalkan HTML untuk membentuk struktur dasar halaman profil dan konten.
    • OpenAI Website (Contoh): Situs web ini, tempat Anda sedang berkomunikasi dengan saya, juga dibangun menggunakan HTML untuk struktur dasarnya
